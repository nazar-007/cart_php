Это корзина, созданная на PHP с использованием JavaScript и JQuery. В качестве базы используется MySQL (cart_goods.sql прикреплен).

1. Заходим в index.php, нажимаем на кнопку "Начать заказ". В базу записывается запись с пока пустым заказом, а после нажатия на кнопку сразу выдается номер заказа, чтобы мы понимали, что заказ уникальный и товары в заказе не повторяются.
2. Выбираем товары, которые также идут из базы. Можем добавлять и отнимать. Справа вверху показывается счетчик - общая сумма заказанных товаров.
3. После этого нажимаем на "Завершить заказ". Сразу открывается модальное окно с заказанными блюдами и итоговой суммой. При закрытии модального окна страница обновляется.
4. При переходе на "Историю заказов" (orders_view.php) мы видим последние 20 заказов. Можем нажать на звездочку над каждым из заказов и смотреть заказанные блюда.
